# 📖 Panduan Penggunaan FFX SENSIX

Ikuti langkah-langkah di bawah ini untuk memastikan aplikasi berjalan dengan optimasi maksimal pada perangkat Anda.

## 1. Persiapan Shizuku
FFX SENSIX membutuhkan layanan **Shizuku** untuk mengakses perintah sistem tanpa root.
- Unduh aplikasi Shizuku di Play Store.
- Aktifkan melalui *Wireless Debugging* (untuk Android 11+) atau via ADB (untuk Android 10 kebawah).
- Pastikan status Shizuku tertulis **"Shizuku is running"**.

## 2. Instalasi FFX SENSIX
- Unduh APK terbaru dari [Halaman Download](https://github.com/cyrust-app/ffxsensix/releases).
- Instal file APK tersebut di ponsel Anda.
- Buka aplikasi, dan ketika muncul jendela pop-up permintaan izin Shizuku, pilih **"Allow all the time"**.

## 3. Menjalankan Optimasi
Di dalam Dashboard FFX SENSIX:
- **Execute Performance:** Gunakan ini sebelum mulai bermain game. Ini akan memicu *High-Performance Mode*.
- **Sensix Scale:** Kami menyarankan Level 3 untuk penggunaan harian, dan Level 5 (VIP) untuk kompetitif.
- **Visual Assist:** Aktifkan jika Anda membutuhkan titik fokus tambahan di layar.

## ❓ FAQ (Pertanyaan Umum)

::: tip Apakah ini aman dari Banned?
Ya, FFX SENSIX bekerja pada level sistem Android (optimasi kernel & UI), bukan memodifikasi file di dalam folder game Free Fire.
:::

::: warning Baterai Cepat Habis?
Mode *Extreme Performance* akan mengonsumsi lebih banyak daya baterai karena CPU dipaksa bekerja pada frekuensi tinggi. Gunakan *Execute Balanced* setelah selesai bermain.
:::
