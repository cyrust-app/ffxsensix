# ⚙️ Sensivitas Dasar

Sebelum mengatur DPI, pastikan pengaturan dasar di dalam game sudah optimal. Pengaturan ini adalah fondasi agar *aim* tidak goyang.

## Pengaturan In-Game (Rekomendasi)

| Item | Nilai | Keterangan |
| :--- | :--- | :--- |
| **Lihat Sekeliling** | 95 - 100 | Untuk pergerakan lincah |
| **Red Dot Sight** | 85 - 90 | Akurasi jarak dekat |
| **2x Scope** | 75 - 82 | Stabil saat *tapping* |
| **4x Scope** | 70 - 75 | Mengurangi *recoil* berat |
| **Sniper Scope** | 50 | Presisi tinggi |

::: tip TIPS
Mulailah dengan nilai **90** untuk semua scope, lalu naikkan atau turunkan **2 poin** setiap kali kamu merasa terlalu lambat atau terlalu cepat.
:::
