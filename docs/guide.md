# Panduan Memulai

Selamat datang di panduan resmi FFXSensix. Ikuti langkah-langkah di bawah ini untuk memulai optimasi perangkat Anda.

## Persyaratan Sistem

::: info INFO
Pastikan perangkat Anda memiliki ruang penyimpanan minimal 50MB bebas.
:::

- Android 10 atau lebih tinggi / iOS 15+
- Koneksi internet stabil
- Akses pengembang (Developer Options) diaktifkan

## Langkah Instalasi

1. **Download:** Unduh versi terbaru dari [Releases](https://github.com/cyrust-app/ffxsensix/releases).
2. **Ekstrak:** Buka file yang telah diunduh.
3. **Konfigurasi:** Sesuaikan nilai DPI sesuai rekomendasi di bagian [Sensivitas](/config/dpi).
